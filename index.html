エレメントを指定して◯◯

ヒットした要素数 : 0

  • id
  • name
  • XPath
  • CSS Selector
driver.findElement(By.id("id_name"));
driver.findElement(By.name("name"));
driver.findElement(By.xpath("//html/body/div"));
driver.findElement(By.css(".class > div:nth-child(1)"));

情報を取得する(複数ある場合は、0番目を対象にする)

  • tagName
  • text
  • size
  • location
driver.findElement(By.id("id_name")).getTagName()
driver.findElement(By.id("id_name")).getText()
driver.findElement(By.id("id_name")).getSize()
driver.findElement(By.id("id_name")).getLocation()
